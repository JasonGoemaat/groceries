# Groceries

This app should be very light-weight and easy to use, letting you create
grocery lists.

## Goals

1. Learn the latest angular version 21
2. Experiment with using AI to help, starting with Gemini
3. Create a useful app that my family can use
4. Learn the SpeechRecognition api

## Initial Ideas

* Have 'push to talk' button, making it super easy to add items
* Initially store lists in localstorage so it can work offline
* Allow sharing of grocery lists
    * Read-only link anyone can use (guid/hash?)
    * Editable link (guid or email?)
* Show most recent lists first, possibly archive them
* Easy to share, click 'share' button to show QRCode and button to copy link
    * This makes it easy to just display and have someone else snap it
* Besides localstorage, can convert to server-side, using PocketBase
